VOCAB RUSH 소개

직장인과 학생을 위한 타이머 기반 학습 챌린지 게임

VOCAB RUSH란?

VOCAB RUSH는 영어 어휘, 수학, 한국어 학습을 게임 형식으로 즐길 수 있는 무료 웹 애플리케이션입니다. 제한 시간 안에 문제를 풀면서 자연스럽게 실력을 향상시킬 수 있습니다.

회원가입 없이 브라우저에서 바로 시작할 수 있으며, PWA(Progressive Web App)를 지원하여 스마트폰 홈 화면에 앱처럼 설치할 수도 있습니다.

학습 모드

📖 영어 어휘 (Vocabulary)

5,000개 이상의 영어 단어를 수준별로 학습합니다.

  • CEFR 기준 A1(기초) ~ C2(최상급) 난이도 분류
  • SAT/GRE 수준의 고급 어휘 포함
  • IT/테크 분야 전문 용어
  • 영어 발음 듣기 + 한국어 뜻 보기 + 4지선다 퀴즈
  • 예문과 힌트 기능으로 문맥 속 학습 가능

🔢 수학 (Math)

다양한 수학 분야의 문제를 풀어보세요.

  • 사칙연산, 분수, 방정식, 기하학 등
  • 프로그래밍 관련 수학(진법 변환, 비트 연산 등)
  • 실생활 응용 문제

🇰🇷 한국어 (Korean)

한국어 어휘력과 문법 실력을 테스트합니다.

  • 맞춤법, 사자성어, 속담
  • IT/비즈니스 용어
  • 올바른 표현 찾기

주요 기능

  • 3단계 난이도 - Easy(15초) / Medium(10초) / Hard(6초)
  • 콤보 시스템 - 3연속 정답 시 보너스 점수
  • 통계 추적 - 누적 정답 수, 최고 점수, 연속 정답 기록
  • 뱃지 시스템 - 다양한 도전 과제 달성
  • 음성 지원 - 영어 단어 발음 듣기 (Web Speech API)
  • 오프라인 지원 - PWA로 인터넷 없이도 플레이 가능

이런 분들에게 추천합니다

  • 출퇴근 시간에 짧게 영어 단어를 외우고 싶은 직장인
  • TOEIC, TOEFL, 수능 영어를 준비하는 학생
  • IT 분야 영어 용어를 익히고 싶은 개발자
  • 한국어 맞춤법과 어휘력을 키우고 싶은 분
  • 수학 기초 실력을 게임처럼 즐기며 키우고 싶은 분

기술 스택

VOCAB RUSH는 최신 웹 기술로 만들어졌습니다.

  • React 18 + Vite (빠른 빌드와 최적화된 성능)
  • PWA (Progressive Web App) - 오프라인 지원
  • Web Speech API - 브라우저 내장 음성 합성
  • 반응형 디자인 - 모바일/태블릿/데스크톱 모두 지원
